Clojure permit redefine macro from inside itself

(def hallo (fn [] (def hallo (fn [] "Hell"))))
(hallo)
(hallo)
First line, i define hallo as function which is will redefine hallo as function which returning string "Hell"
Second line, i invoke hallo and it will redefine itself
Third line, i invoke hallo and it return string "Hell"

It is interesting because clojure permitted this behavior while some language not.
