What is MD5

It is one of hashing techniques

Naive implementation

based on RFC 1321, peek and copy from Github, and pseudocode at Wikipedia". Code in Racket could be found here. This implementation cannot processing message bigger than provided memory because it is trying to put whole message into memory.

Better implementation with Port

The algorithm only need processing 64 bytes block each time, then we doesn't need all data in memory. We only need to customise input port to include extra padding and message length.
